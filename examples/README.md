# Examples

These scenarios are designed to make The Order of the Agents' value obvious quickly.

Run one with:

```bash
agent-order ./examples/review-agent-order-readme/scenario.md --agents codex,claude
```

Recommended first examples:

- `review-agent-order-readme`: self-review the launch surface.
- `build-vs-buy-analytics`: practical product/engineering tradeoff.
- `rest-to-trpc`: architecture decision with real constraints.

Future launch examples should include complete rendered run folders with `index.md`, `final/report.md`, `final/decision-log.md`, and selected `turns/*.md`.
